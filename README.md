# 🌦️ Polymarket Weather Bot

Автоматический бот для ставок на погодные рынки [Polymarket](https://polymarket.com).

Идея проста: Open-Meteo даёт точный прогноз погоды бесплатно. Polymarket — prediction market, где цены часто неэффективны. Если наш прогноз расходится с ценой рынка на >5% — делаем ставку.

## Архитектура

```
weather.py   ← прогноз погоды (Open-Meteo, бесплатно, без API ключа)
markets.py   ← находит погодные рынки на Polymarket
strategy.py  ← сравнивает наш прогноз vs цена рынка, считает Kelly
bot.py       ← main loop, управляет ставками
```

## Быстрый старт

```bash
# 1. Установить зависимости
pip install -r requirements.txt

# 2. Настроить .env
cp .env.example .env
# вставить свой Polymarket private key

# 3. Запустить в dry run (без реальных денег)
python bot.py

# 4. Запустить в боевом режиме
python bot.py --live

# 5. Запустить в цикле каждый час
python bot.py --live --loop
```

## Сигналы

| Сигнал | Логика |
|--------|--------|
| `rain` | Вероятность осадков по Open-Meteo + количество мм |
| `heat` | Макс. температура vs порог (по умолчанию 35°C) |
| `cold` | Мин. температура vs порог |
| `snow` | Вероятность снега = холод × осадки |
| `wind` | Скорость ветра / 80 км/ч |

## Стратегия

- **Edge**: ставим только если наша вероятность отличается от рыночной на ≥5%
- **Размер**: половина Kelly criterion, максимум `MAX_BET_USDC` за ставку
- **Города**: New York, London, Tokyo, Miami, Paris

## Настройки (.env)

| Параметр | По умолчанию | Описание |
|----------|-------------|----------|
| `DRY_RUN` | `true` | Симуляция без реальных денег |
| `MAX_BET_USDC` | `5` | Максимальная ставка в USDC |
| `MIN_EDGE` | `0.05` | Минимальный edge для ставки (5%) |

## ⚠️ Дисклеймер

Это образовательный проект. Polymarket — prediction market, не биржа. Торговля связана с риском потери средств. Используйте на свой страх и риск.
